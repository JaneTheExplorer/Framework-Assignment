# Frameworks_Assignment: CORD-19 Data Explorer

## Overview
This project explores the CORD-19 metadata dataset, focusing on COVID-19 research papers.  
It includes data cleaning, analysis, visualization, and a Streamlit app for interactive exploration.

## Features
- Load and clean metadata.csv
- Analyze publications by year, journals, and sources
- Generate word cloud of paper titles
- Interactive Streamlit app with filters and charts
